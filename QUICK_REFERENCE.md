## 🚨 TROUBLESHOOTING

### "Can't login"
```
Check: URL is exactly correct
Check: PIN is recent (not old)
Try: Hard refresh (Ctrl+Shift+R)
Try: Incognito mode
Last: Clear browser data
```

### "Portal forms don't show"
```
Check: URL has "?mode=portal" at end
Check: Not in admin mode (need to be logged out)
Try: Different browser
Try: Incognito/private mode
```

### "Data not saving"
```
Check: Browser allows local storage
Try: Different browser
Try: Hard refresh (Ctrl+Shift+R)
Last: Clear cache and try again
```

### "Lost my PIN"
```
Option 1: Restore from backup
   - Download backup file
   - Restore in Settings
   - Start fresh (PIN reset)

Option 2: Clear data
   - Settings → Clear All Data
   - Restore from backup
   - Login with old PIN
```
