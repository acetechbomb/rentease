1. Click Download Backup File — a file saves to your Downloads folder.
2. Share that file to WhatsApp, Gmail, or cloud storage for safekeeping.
3. To restore: open this page on any device → Backup tab → Choose Backup File → select the .json file.
4. All your data will be restored instantly!
