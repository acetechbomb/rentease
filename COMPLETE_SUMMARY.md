## 🛠️ Common Tasks

### Change PIN
1. Open app (log in with old PIN)
2. Click ⚙️ (Settings)
3. Enter new 4-digit PIN
4. Click "Update PIN"
5. Logout and test new PIN

### Add a Property
1. Go to "Properties" tab
2. Click "+ Add Property"
3. Fill: Name, type, location, beds, status, rent
4. Click "Save Property"
5. ✓ Done - appears in your list

### Register a Tenant
1. Go to "Tenants" tab
2. Click "+ Add Tenant"
3. Fill: Name, phone, email, property, rent, move-in date
4. Click "Save Tenant"
5. ✓ Done - ready for payment tracking

### Record a Payment
1. Go to "Payments" tab
2. Click "+ Record Payment"
3. Select: Tenant, property, month, amount, status, date
4. Click "Save Payment"
5. ✓ Done - shows in dashboard

### Add an Expense
1. Go to "Expenses" tab
2. Click "+ Add Expense"
3. Fill: Description, property, category, amount, date, notes
4. Click "Save Expense"
5. ✓ Done - tracked for accounting

### Download Backup
1. Click ⚙️ (Settings)
2. Click "📥 Download Backup"
3. File saves (rentease-backup-YYYY-MM-DD.json)
4. Keep it safe (Google Drive, Dropbox, etc.)

### Restore from Backup
1. Click ⚙️ (Settings)
2. Click "📤 Restore from Backup"
3. Select your backup file
4. Confirm warning
5. ✓ All data restored

## ❓ FAQ

**Q: Does it cost anything?**
A: No! GitHub pages is free forever. Optional custom domain is $10-15/year.

**Q: Is my data safe?**
A: Yes! Your data stays on your device. No servers access it. You control backups.

**Q: Can tenants hack it?**
A: No PIN needed for portal, but they can only submit data (you approve). Hacking gives them no access to other tenants' data.

**Q: What if I forget my PIN?**
A: Download a backup file, edit it with a text editor to reset PIN, then restore. Or clear browser data and start fresh.

**Q: Can I use on mobile?**
A: Yes! Full mobile support. Can install as app on Android/iPhone home screen.

**Q: What if GitHub goes down?**
A: Unlikely (99.9% uptime), but you have backups. Download weekly to be safe.

**Q: Can I edit my app after uploading?**
A: Yes! Make changes anytime in your GitHub repo. Updates live in 2-3 minutes.

**Q: How do I add my logo?**
A: Edit index.html in GitHub web editor. Find the 🏠 emoji and replace with your logo text/image.

**Q: Can multiple people access?**
A: Currently one admin (PIN protected). Could add co-managers with shared PIN.

**Q: What happens to old data?**
A: You keep all backups. Data stays on your device until you delete it.

**Q: Is there a limit to how many properties/tenants?**
A: No limit! Browser storage supports thousands of records.
