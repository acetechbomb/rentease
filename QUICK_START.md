# ⚡ RentEase GitHub Deployment - QUICK START

## In 5 Steps 🚀

### 1️⃣ CREATE GITHUB ACCOUNT
```
Visit: https://github.com
Click: Sign Up
Verify email
```

### 2️⃣ CREATE REPOSITORY
```
Click: + (top right)
New repository
Name: rentease
Choose: Public
Create repository
```

### 3️⃣ UPLOAD FILES
```
Click: Add file → Upload files
Select:
  ✓ index.html
  ✓ manifest.json
  ✓ sw.js
  ✓ README.md
Click: Commit changes
```

### 4️⃣ ENABLE GITHUB PAGES
```
Go to: Settings
Find: Pages
Source: Deploy from a branch
Branch: main
Folder: / (root)
Click: Save
```

### 5️⃣ GET YOUR LINK ✨
```
Wait 2-3 minutes...

Your site is published at:
https://YOUR_USERNAME.github.io/rentease
```

## 🚨 COMMON ISSUES

| Problem | Solution |
|---------|----------|
| "Not found" | Wait 3 min, check repo name in URL |
| Portal not working | Check URL has `?mode=portal` |
| Changes not showing | Hard refresh: Ctrl+Shift+R |
| PWA offline broken | Check sw.js is in root |
| Files not showing | Go to Settings → Pages → Verify source |
